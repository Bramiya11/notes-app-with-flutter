# App de Notas Personales con Flutter

## ¿Qué es esto?
Más que un proyecto de aula para Desarrollo Móvil, este proyecto constituye para nosotros una experiencia grata en el trabajo y la colaboración en equipo.
Nuestra App nace como una necesidad de tener un entorno que pueda sentirse coherente en los dos sistemas operativos móviles más utilizados (***Android - iOS***), ya que como sabemos, cada sistema operativo tiene una App de notas predeterminada muy diferente, lo que afecta un poco la experiencia del usuario a la hora de cambiar de dispositivo.
Es ahí donde entra nuestra idea: con **Flutter**, crear una experiencia por igual a cada usuario, independientemente de su sistema operativo o dispositivo.

## Instalación y ejecución de la App ***(en este caso, a dispositivos de escritorio)***
1. Clonar el repositorio a una carpeta local, preferiblemente a una carpeta de repositorios fuera de OneDrive (*C:\Repositorios*).
```bash
git clone https://github.com/Bramiya11/notes-app-with-flutter.git
```
2. Entrar al proyecto e instalar dependencias.
```bash
cd notes-app-with-flutter
flutter pub get
```
3. Ejecutar la aplicación.
```bash
flutter run
```
